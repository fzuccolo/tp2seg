# TecnoHogar

Dataset del caso principal del TP2. En esta etapa todavia es una semilla parcial para mantener funcionando el pipeline; en la etapa 3 se reemplaza por datos completos y defendibles.

## Archivos actuales

- `empresa.yml`: metadata basica de TecnoHogar S.A.
- `activos.csv`: activos y procesos derivados del caso trabajado en TP1.
- `diagnostico.csv`: controles iniciales evaluados con escala CMMI.
- `proyectos.csv`: cartera preliminar de mejoras.
- `proyecto_control.csv`: trazabilidad inicial entre controles y proyectos.

## Proxima etapa

Completar los 93 controles ISO 27002:2022 con evidencia, madurez, entrevistado/fuente, brecha y observaciones; despues ajustar proyectos, pesos y priorizacion para que el informe profesional salga de datos reales del caso.
